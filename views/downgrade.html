In TLS the random number field in the serverHello prevents downgrade attack.

If the client's version is 1.2 and the server 1.3, the random field value in the serverHello should be equal to:
44 4F 57 4E 47 52 44 01

If the client's version is 1.1 or below and the server is 1.3 or 1.2 the random field value in the serverHello should be equal to:
44 4F 57 4E 47 52 44 00

In this example both the Client and Server are TLS1.3.
